# Bank Telegram Bot

Проект предоставляет инструменты для 
управления пользовательскими данными и созданием удобных денежных переводов.

## Список команд
* `/start` — запуск бота.
* `/restart` — перезапуск бота.
* `/register` — зарегистрироваться.
* `/edit` — редактировать информацию о себе.
* `/delete` — удалить аккаунт.
* `/help` — вывод общей информации о боте и списка команд.

## Стек технологий
| Технология                  | Описание                                             |
|:----------------------------|:-----------------------------------------------------|
| Java 21                     | Язык программирования                                |
| Spring Boot 3+              | Фреймворк для разработки приложения                  |
| PostgreSQL                  | Основная БД для хранения профилей и истории операций |
| Spring Data JPA / Hibernate | Инструменты для работы с базой данных                |
| TelegramBots API            | Библиотека для взаимодействия с Telegram             |
| Lombok                      | Библиотека для сокращения шаблонного кода            |
| SLF4J                       | Логирование всех событий и ошибок                    |

## Основные возможности
* **Регистрация:** Создание профиля пользователя с привязкой ID и хэшированием номера телефона (SHA-256).
* **Управление профилем:** Возможность редактирования личной информации и полного удаления аккаунта и всех связанных с ним данных из БД.
* **FSM (Finite State Machine):** Надежное управление состоянием диалога каждого пользователя через `ConcurrentHashMap`.

## Технические ограничения
В текущей реализации предусмотрена логическая заглушка для совершения перевода. Прямая интеграция (deeplinks) сталкивается с ограничениями безопасности:
* **Блокировка протоколов:** Прямые вызовы банковских схем (tbank://, и т.д.) блокируются Telegram API.
* **Безопасность ОС:** Автоматический переход в банковские приложения через промежуточные Web-страницы блокируется системами безопасности Android.

## Запуск проекта

Для запуска бота в локальной среде необходимо выполнить следующие шаги:

### 1. Подготовка окружения
Убедитесь, что у вас установлены:
*   Java 21
*   PostgreSQL

### 2. Сборка проекта
Для компиляции проекта выполните команду в корневой директории:
```bash
# Для Windows
.\mvnw.cmd clean package -DskipTests

# Для Linux / macOS
./mvnw clean package -DskipTests
```

### 3 Запуск
Приложение требует DB_PASSWORD (пароль от БД) и BANK_BOT_KEY (токен бота).

#### Способ А
Установите переменные в вашей ОС и запустите JAR:
```bash
java -jar target\[ИМЯ_ВАШЕГО_ФАЙЛА].jar
``` 

#### Способ Б
Если переменные не заданы глобально, используйте команду:
```bash
$env:DB_PASSWORD='ПАРОЛЬ_ОТ_БД'; $env:BANK_BOT_KEY='ТОКЕН'; java -jar target\[ИМЯ_ВАШЕГО_ФАЙЛА].jar
```
